# Doc-Genie

Doc-Genie is a Python-based tool that automatically generates docstrings 
for Python functions and classes.

Built using:
- Python
- Gradio
- ReportLab

## How to Run

1. Install dependencies
pip install -r requirements.txt

2. Run the program
python doc_genie.py

## Features
- Upload Python file
- Generate docstrings
- Export as TXT or PDF
- Share links
